基本操作

移动

光标上移 ：k

光标左移 ：h

移动到行首 ：0

移动到行首词 ：^

光标下移 ：j

光标右移 ：l

移动到行尾：$

数字键 + 移动键，比如： 6j ， 代表向下移动6行

3- / 3+ 分别代表上移、下移3行行首

3$ / 3$ 分别代表上移、下移3行行尾

  • 移动到下一个词的词首 w
  • 移动到下一个大词的词首 W
  • 移动到下一个词的词尾 e
  • 移动到下一个大词的词尾 E
  • 移动到上一个词的词首 b
  • 移动到上一个大词的词首 B
  • 选中1整个单词 ：bve （可视化的选择）

跳转

  • 跳转到文件第一行 ： gg

  • 跳转到文件最后一行： G

  • 上下翻页 ：Ctrl + u；Ctrl + d

  • 上下滚动 ：Ctrl + e；Ctrl + y

删除、修改

  1. 删除一个字符： x
  2. 删除一个词： dw 、 bvex（可视化的删除）
  3. 删除一个词，并等待输入 ： cw = dwi（也可以选中单词，再c）
  4. 删除一个词，并等待输入 ： ciw 或 ciW

缩进

  1. 换行缩进 :set cindent
  2. 设置tab键显示大小 :set ts=4 :set sw=4 :set et noet
  3. V(行选中) = 自动缩进、» 、>i {
  4. 设置filetype on filetype plugin on filetype indent on
  5. vimrc里输入“来注释

插入

  1. I 行首（^）插入
  2. i 当前字符前边插入（insert）
  3. a 当前字符后边插入（append）
  4. A 行尾插入
  5. 在网站复制的文字，进入vim，先i，用Shift + Insert 粘贴
  6. 交换两行 ddp

可视化模式

  1. q + u（宏的名称） + 操作：（xj）→ @ + u (调用刚才保存在u的操作)（4 + @ + u 或4 + @ + @ ）
  2. v/V + j（选中行） + : norm 0x（跳到行首，删除一个字符）+Enter

查找

  1. / + ”想要查找的单词” → n 下一个、N 上一个
  2. ？+ ”想要查找的单词” → n 上一个、N 下一个
  3. 光标停在某一个单词上，按 # → 向下查找(同 1 )，可以先bve，viw
  4. 光标停在某一个单词上，按 * → 向上查找(同 2 )，可以先bve，viw
  5. f + “想要查找的单词” ，自动跳转到此单词处，按 ；下一个
  6. 结合5，可使用 c+f+字符 ，将 字符 到行首的内容全部删掉，并等待输入
  7. t + “想要查找的单词” ，自动跳转到此单词前一位，按 ；下一个
  8. 结合5，可使用 c+t+字符 ，将 字符前 到行首的内容全部删掉，并等待输入

替换

  1. :s/单词1/单词2 （只换第一个）
  2. :s/单词1/单词2/g （多次替换、全换）
  3. :%s/单词1/单词2/g （全换）
  4. :%s/usr/lib/lib/g （usr/lib全换成lib）
